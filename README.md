# DailyTasks App

---

## Built with

- Nuxt 3 and Vue.js

---

#### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/amt-daniel-adeabah/dailytasks-app.git
    ```

2. Navigate to the project directory:
    ```bash
    cd dailytasks-app
    ```

3. Install the dependencies:

    Using npm:
    ```bash
    npm install
    ```

    Using yarn:
    ```bash
    yarn install
    ```

4. Start the development server:

    Using npm:
    ```bash
    npm run dev
    ```

    Using yarn:
    ```bash
    yarn dev
    ```
